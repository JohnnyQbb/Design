Intro

這一份 Design Guideline 為了幫助在各個服務之間建立容易遵守的規範，並減少開發與設計成本，以下會逐一介紹如何使用各種設計元素的設計系統、貢獻以及維護方式，期待讓產品開發可以更便捷與輕鬆，讓設計們事半功倍。

Goal

  • 1.降低每個組員介入項目的門檻成本。
  • 2.提高工作效率及協同開發的便捷性。
  • 3.高度統一的風格。
  • 4.提升代碼質量和可維護性。
  • 5.讓 RD 也能快速做出高質量的符合需求的頁面。

Setup

在要編輯的頁面的最上面， @use 這兩個 scss

請參考 _component_library

                                
    //讀取樣式//
    @use "BH-style" as *;
    
    //讀取元件//
    @use "BH-component" as *;

Color Data-Theme

此文件內部的所有元件、文字、匡線、背景都已經套用變數來讓 RD 可以一鍵切換亮暗色版本，理論上不再需要也不應該多寫一支 -dark.scss 的檔案，提升切版效，點擊右上角的開關可以預覽切換。

請參考 _color.scss

                            
    /*example*/                            
    color: var(--primary);

Font-Size Definition

h 與 p 標籤都已經內建預設的大小、文字顏色，以及 line-height 預設值，並預設可以用 RWD 同步縮放字體的功能，不受瀏覽器內建的縮放影響。

請參考 _font.scss

                             
                /載入    /字級     /行高                                
    <p class="paragraph-small-spacing-m"></p>

Spacing System

透過將所有間距定義為 4 的倍數，並且利用迴圈將所有排列組合產生出 class ，可以直接用增加以及調整 class 的方式去設定 paddiong & margin & gap 這三個屬性，大幅減少 scss 的行數與撰寫量。

請參考 _spacing.scss 以及嘗試更改帶有 pd mg gp 這三個開頭的class

                            
    <p class="pd12"></p>
    
    <p class="pd12-pd8"></p>
    
    <p class="pd12-right"></p>

Z-Layers System

透過建立 map 與 map-get 語法，將 z-index 的值給予直觀的定義，來避免層級混亂。

請參考 _z-index.scss

                                
    /*example*/            
            函式呼叫    （$key）                        
    z-index: z-layers(default);
    
    /map
    $z-layers: (
      "popup":                1000,
      "fix":                  100,
      "nav":                  10,
      "cover":                2,
      "default":              1,//預設值
      "below":                -1,
      "hidden":               -10
    );
    //縮寫
    @function z-layers($key) {
      @return map-get($z-layers, $key);
    }

Autolayout System

透過類似 bootstrap 串接 class 名稱的方式，將 figma autolayout 的模式轉換為 class 讓設計可以與切版更加一致，也減少為了排版而產生的一次性 class

請參考 _frame.scss

                            
                    載入    /啟動flex    /方向  /背景 
    <div class="BH-frame-autolayout-column-f1">
                                        /對齊
        <div class="BH-frame-autolayout-align_center-justify_center"></div>                
    </div>

OOCSS

此文件也將 html 架構與元件完全剝離，不會被過深的架構影響而難以覆蓋、沿用、維護

Prefix Class Name

此文件內的所有元件 class name 都帶有 prefix 的前綴設定，避免因為使用 attr selector 造成容易衝突的問題，

請參考 /BH-component

                            
    /*example*/            
    .#{$prefix}-example {
        display: flex;
    }
